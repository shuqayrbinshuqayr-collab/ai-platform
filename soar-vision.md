# SOAR.AI - Vision Document (from PDF)

## Brand
- Name: SOAR.AI
- Tagline: AI-Powered Design
- Colors: Dark background (near black), Orange accent (#FF6B00 approx), White text
- Logo: House/building icon with "SOAR" text

## User Flow (4 Steps)

### Step 0: Welcome Screen - Choose Action
- 3 options: Create Plan | Create Image | Create Project (Pro - locked)
- If "Create Plan" → ask: Villa or Apartment building?

### Step 1: AI Land Analysis
- Two input methods:
  1. Google Maps integration - owner pins land location on map
  2. Enter "Sak" (صك) number - land deed number to auto-fetch land data
- Output: AI Land Analysis Report showing:
  - Land area, zoning (residential/commercial), coordinates
  - Design Code Compliance score (e.g. 92%)
  - Max Height, Setbacks, FAR, Garage placement
  - Municipality regulations, Urban code, Local conditions

### Step 2: Data Entry Form
- After land is identified, show form with:
  - Building Type tabs: Residential | Commercial | Mixed-Use
  - Number of floors
  - Land area
  - Number of rooms
  - Number of bathrooms (دورات المياه)
  - Number of kitchens (مطابخ)
  - Number of dining rooms (غرف الطعام)
  - Number of living rooms (صالات)
  - Number of majlis (مجالس)
  - Number of maid rooms (غرف الخدم)
  - Laundry room (غرفة الغسيل)
  - Number of driver rooms (غرف السائقين)
  - Number of annexes (ملاحق)
  - Number of elevators (مصاعد)
  - Number of parking spaces (مواقف السيارة)
  - Number of entrances (مداخل)
  - Number of garages (كراجات)
  - Number of club rooms (غرف النادي)
  - Number of balconies (بلكونات)
  - Choose facade style from ready-made facades
  - Blueprint built based on chosen facade

### Step 3: Generate Blueprints
- Generate 6 different blueprint concepts (Concept 1-7)
- Each concept shows floor plan with room labels and dimensions
- User can select preferred concept
- Option to add second floor for each concept
- "+Add" button per concept

### Step 4: Download
- Download blueprints as:
  - Revit format
  - AutoCAD format
  - Or request review & official certification from SOAR team

## Key Insights
- This is a B2C platform (direct to homeowners/clients), not just B2B engineering offices
- The platform generates MULTIPLE concepts (6+) for the user to choose from
- Google Maps integration is critical for land identification
- "Sak" number (صك) input for Saudi land deed system
- Facade selection influences the floor plan layout
- Professional certification/review service offered by SOAR team
- The design aesthetic is DARK + ORANGE (not blue blueprint style)
